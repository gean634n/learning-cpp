## Pré-requisitos para Rodar os Exemplos 🛠️

1. **Ubuntu**:
   - GCC instalado: `sudo apt install g++`.
   - Editor recomendado: [Visual Studio Code](https://code.visualstudio.com/).

2. **Compilar e rodar um exemplo**:
   ```bash
   g++ arquivo.cpp -o programa
   ./programa
   ```
